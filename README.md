# NPM INSTALL

npm i dropdownlib-reda

## IMPORT DROPDOWN

import {DropDown} from 'dropdownlib-reda'

### USE DROPDOWN
 You have 4 params in the function
 1. label
 2. options
 3. handleSelect
 4. id
 You need, in your options, an array of object ex: options={[{}]}, that's all !
